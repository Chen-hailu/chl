An open-source MATLAB implementation for correcting stratified tropospheric delays in InSAR using an iterative weighted least-squares approach. The code can be directly applied to MintPy HDF5 outputs. The functional model is flexible and can be adapted to specific requirements, such as higher-order polynomial representations or block-based estimation strategies.
